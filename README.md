# AI Resume Parser

A lightweight AI-powered resume parser that extracts structured information from PDF resumes using LLaMA (via Groq API) and displays it through a simple Streamlit web app.


## 💡 About the Project

This project demonstrates how large language models can be used to convert unstructured resume data into structured, usable information. It combines traditional PDF text extraction with AI-based parsing to create a simple yet effective resume analysis tool.


## 🚀 Features

- 📄 Upload PDF resumes  
- 🧠 Extracts:
  - Name, Email, Phone  
  - Skills  
  - Education  
  - Experience  
  - Summary  
- ⚡ Uses LLaMA 3 (via Groq) for intelligent parsing  
- 📊 Returns clean structured JSON  
- 💻 Simple Streamlit UI for easy interaction  


## 🛠️ Tech Stack

- Python  
- Streamlit (Frontend UI)  
- PyMuPDF (fitz) (PDF text extraction)  
- Groq API (LLaMA 3.3) (AI parsing)  
- python-dotenv (API key management)
